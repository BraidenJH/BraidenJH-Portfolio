# Braiden's Portfolio / Landing Page

This is my personal website to showcase my apps, games, and web projects. Built with **HTML, CSS and a little bit of JavaScript.**

---

## Key Features

- **Works on all screen sizes and devices** – The layout uses Flexbox and CSS Grid, so it adapts to whatever device is being used. Also uses media queries to handle adjustments and toggle specific elements at certain breakpoints, like showing a hamburger menu icon on smaller screens instead of a full navigation bar. 

- **Clean, modern design** – Features a clean layout with a gradient used across buttons, text highlights and the scrollbar. Along with smooth hover animations and colour changes throughout.
