# 🌟 Braiden's Portfolio / Landing Page

A personal portfolio website showcasing my apps, games & web projects. Built with **HTML, CSS & JavaScript**, designed for performance, responsiveness, and interactivity.  

---

## 🚀 Features

- 🌐 **Responsive layout for all devices** – Built using **Flexbox and CSS Grid**, the layout adapts seamlessly to desktop, tablet, and mobile screens. Media queries dynamically adjust content flow, font sizes, and image scaling.  

- 🖱 **Smooth scrolling navigation** – Anchored links with **CSS `scroll-behavior: smooth`** create fluid transitions between sections without JavaScript, improving user experience.  

- 💻 **Stylised & functional buttons** – Custom buttons use **JavaScript functions** to open external links in new tabs (`window.open()`) while maintaining **security with `noopener` and `noreferrer`**. Gradient backgrounds, hover scaling, and smooth transitions make them visually appealing and interactive.  

- 🎨 **Clean and modern design** – Minimalistic layout, consistent typography (**Poppins**), gradient accents, and subtle hover animations deliver a polished, professional look.  

- 📱 **Optimised for mobile, desktop, and tablet** – Media queries fine-tune layout, spacing, image dimensions, and navigation behaviour to ensure excellent usability across all device types.
